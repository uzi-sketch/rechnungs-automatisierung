# Rechnungs- & Mahnungs-Automatisierung

**Automatische Erstellung professioneller Rechnungen mit Mahnungsfunktion für kleine und mittlere Unternehmen (KMU).**

Dieses Tool wurde entwickelt, um Selbstständigen und kleinen Unternehmen Zeit bei der Rechnungsstellung und dem Mahnwesen zu sparen.

## Features

- Automatische Erstellung von Rechnungen als PDF
- Mehrere Rechnungspositionen mit automatischer Summenberechnung
- Direkter Download einzelner Rechnungen
- "Alle Rechnungen als ZIP" herunterladen
- Interaktive Web-Oberfläche (Streamlit)
- Mahnungs-Simulation für überfällige Rechnungen
